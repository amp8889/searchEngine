Template documentation[view] [edit] [history] [purge]

Usage

This template's initial visibility currently defaults to autocollapse, meaning that if there is another collapsible item on the page (a navbox, sidebar, or table with the collapsible attribute), it is hidden apart from its title bar; if not, it is fully visible.

To change this template's initial visibility, the |state= parameter may be used:

  • {{Basic information|state=collapsed}} will show the template collapsed, i.e. hidden apart from its title bar.
  • {{Basic information|state=expanded}} will show the template expanded, i.e. fully visible.

Purposes

Similar to the Template:Wikipedia help pages, but simplified. But all the links of that template should be able to be accessed if the links at the top of the navbox are accessed (i.e. Help Menu; Tutorials and introductions; The Missing Manual; Noticeboards; FAQs; Reference desk). In addition, some extra links from Category:Wikipedia basic information are included, as well as some other links. Also, the links are rearranged into different groups, which should be self-explanatory.

  • Help menu: the 13 links are not included, but the Help menu link is at the top of the navbox.
  • Tutorials and introductions: this is included at the top of the navbox. Most tutorials and introductions are included in the navbox, but they are included within the other groups, e.g. [[Help:Navigation|Navigation]] ([[Help:Introduction to navigating Wikipedia/1|Introduction]]), producing Navigation (Introduction).
  • The Missing Manual: the 25 links from the book are not included, but there is a link for the book contents at the top of the screen.
  • FAQs: the numerous FAQs (apart from the readers' one) are not included, but there is a link to the FAQ index at the top.
  • Noticeboards: most of the noticeboards are not listed, but there is a link at the top to a page, to a page with a navbox listing them all.
  • Reference desk: the reference desk subjects are not listed, but there is a link at the top of the navbox, to a page listing them all.

In addition:

  • Additional links are included, including some from Category:Wikipedia basic information.
  • Links are rearranged into different groups.
