Articles undergoing major edits[edit]

Please note that these templates are courtesies; nobody is required to stop editing just because they're shown.

What to type What it makes
{{In use}}
links talk edit
This project page is actively undergoing a major edit for a short while. To help avoid edit conflicts, please do not edit this page while this message is displayed.
This page was last edited at 21:46, 21 May 2023 (UTC) (10 months ago) – this estimate is cached, update. Please remove this template if this page hasn't been edited for a significant time. If you are the editor who added this template, please be sure to remove it or replace it with {{Under construction}} between editing sessions.

the information in the History section}}

the information in the History section. To help avoid edit conflicts, please do not edit this page while this message is displayed.

If you are the editor who added this template, please be sure to remove it or replace it with {{Under construction}} between editing sessions.
{{Under construction}}
links talk edit
This project page or section is in a state of significant expansion or restructuring. You are welcome to assist in its construction by editing it as well. If this project page has not been edited in several days, please remove this template.
If you are the editor who added this template and you are actively editing, please be sure to replace this template with {{in use}} during the active editing session. New articles[edit]

What to type What it makes
{{In creation}}
links talk edit
This page is currently being created
It is a new Wikipedia article developing through collaboration and successive edits.
To help avoid edit conflicts and other confusion, the creator asks that for a short time this page not be edited unnecessarily, or nominated for deletion during this early stage of development.
  • Please remove this notice if the page has not been edited for several hours.
  • This page was last edited by WhatamIdoing (talk | contribs) 10 months ago. (Update timer)
{{Newlist}}
links talk edit
This list has not been completed to the author's plans. The time-consuming process may be ongoing. You are welcome and encouraged to assist. If you have any questions, please contact the author or leave a message on this list's talk page.
Please do not rush to have it deleted unless it clearly fails to meet Wikipedia's inclusion guidelines. Check back in a while and/or contact the creator to ascertain their plans. If the creator has abandoned work on this page and is not responding to communication, you may consider deletion or userfy it to the creator's userspace (ie User:Creator's username/This page's title).
This page was last edited by WhatamIdoing (talk | contribs) 10 months ago. (Update timer)

Protected articles, pages and images[edit]

Please note that these templates are only meant to indicate that pages have been protected; adding any of them to a page will not add any protection to that page.

What to type What it makes
{{pp-dispute}}
links talk edit
This page is currently protected from editing until editing disputes have been resolved.
This protection is not an endorsement of the current version. See the protection policy and protection log for more details. Please discuss any changes on the talk page; you may submit an edit request to ask an administrator to make an edit if it is uncontroversial or supported by consensus. You may also request that this page be unprotected.
{{pp-vandalism}}
links talk edit
Editing of this page by new or unregistered users is currently disabled due to vandalism.
See the protection policy and protection log for more details. If you cannot edit this page and you wish to make a change, you can submit an edit request, discuss changes on the talk page, request unprotection, log in, or create an account.
{{pp-vandalism}}
links talk edit
This page is currently protected from editing due to vandalism.
This protection is not an endorsement of the current version. See the protection policy and protection log for more details. Please discuss any changes on the talk page; you may submit an edit request to ask an administrator to make an edit if it is uncontroversial or supported by consensus. You may also request that this page be unprotected.
{{pp-protected}}
links talk edit
Editing of this page by new or unregistered users is currently disabled.
See the protection policy and protection log for more details. If you cannot edit this page and you wish to make a change, you can submit an edit request, discuss changes on the talk page, request unprotection, log in, or create an account.
{{pp-protected}}
links talk edit
This page is currently protected from editing.
See the protection policy and protection log for more details. Please discuss any changes on the talk page; you may submit an edit request to ask an administrator to make an edit if it is uncontroversial or supported by consensus. You may also request that this page be unprotected.
{{pp-sock}}
links talk edit
Editing of this page by new or unregistered users is currently disabled to prevent sock puppets of blocked or banned users from editing it.
See the protection policy and protection log for more details. If you cannot edit this page and you wish to make a change, you can submit an edit request, discuss changes on the talk page, request unprotection, log in, or create an account.
{{pp-move}}
links talk edit
{{pp-office}}
links talk edit
This page is currently under the scrutiny of the Wikimedia Foundation Office and is protected.
If you can edit this page, please discuss all changes and additions on the talk page first. Do not remove protection from this page unless you are authorized by the Wikimedia Foundation to do so.

Copyright violations[edit]

What to type What it makes
{{Copyvio|url=http://example.org}}
Talk
{{Copyvio-revdel}}
links talk edit
Certain historical revisions of this page may meet criterion RD1 for revision deletion, as they contain significant copyright violations Error: No URL has been provided! that have been removed in the meantime.
  • The revisions to be redacted have not been specified. Please check the talk page for more information.

Note to admins: In case of doubt, remove this template and post a message asking for review at WT:CP.

Note to the requestor: Make sure the page has already been reverted to a non-infringing revision or that infringing text has been removed or replaced before submitting this request. This template is reserved for obvious cases only, for other cases refer to Wikipedia:Copyright problems.

Note to others: Please do not remove this template until an administrator has reviewed it.
{{Copypaste}}
links talk edit
This article or section may have been copied and pasted from another location, possibly in violation of Wikipedia's copyright policy. Please review the source and remedy this by editing this article to remove any non-free copyrighted content and attributing free content correctly, or flagging the content for deletion. Please be sure that the supposed source of the copyright violation is not itself a Wikipedia mirror.
{{Close paraphrasing}}
links talk edit
This article contains close paraphrasing of non-free copyrighted sources. Relevant discussion may be found on the talk page. Please help rewriting it with your own words. (Learn how and when to remove this template message)
{{Copyvio links}}
links talk edit
This article or section may contain links to copyright infringing works. Please edit this article to remove links to such works.

Articles requiring significant attention and other issues[edit]

What to type What it makes Where it goes
{{Multiple issues}}

Edit – Talk – Links

This article has multiple issues. Please help improve it or discuss these issues on the talk page. (Learn how and when to remove these template messages)

No issues specified. Please specify issues, or remove this template.

(Learn how and when to remove this template message)
  • Use the section=y parameter if you use the template for a particular section.
  • To tag a specific issue, set any string of text to the parameter, such as issue=y
  • To specify the month tagged for the categories that sort articles by month, type issue=April 2024 or issue=April 2007
 Article or section with three or more issues
{{Expert needed}}

Edit – Talk – Links – Category

This article needs attention from an expert in Sample. Please add a reason or a talk parameter to this template to explain the issue with the article. WikiProject Sample may be able to help recruit an expert.
 Article page
{{Globalize}}

Edit – Talk – Links – Category

The examples and perspective in this article may not represent a worldwide view of the subject. You may improve this article, discuss the issue on the talk page, or create a new article, as appropriate. (Learn how and when to remove this template message)
 Article page
{{Missing information}}

Edit – Talk – Links – Category

This article is missing information about some issue. Please expand the article to include this information. Further details may exist on the talk page.
 Article page
{{ISSN-needed}}

Edit – Talk – Links – Category

This article refers to a periodical that doesn't have its ISSN information listed. If you can, please provide it.
 Talk page
{{Very long}}

Talk – Links – Category

This page may be too long to read and navigate comfortably. Consider splitting content into sub-articles, condensing it, or adding subheadings. Please discuss this issue on the article's talk page.
 Article page
{{Citations broken from PEIS limit}}
Talk – Links – Category
This page exceeds Wikipedia's template size limit. Some references and templates may not render properly on this page.
 Article page

Categorization issues[edit]

This is about categorization issues in articles. For tags that are placed on categories, see Wikipedia:Template messages/Category namespace and Wikipedia:Template messages/Deletion.

What to type What it makes
{{Uncategorized}}
category links talk edit
This page has not been added to any categories. Please help out by adding categories to it so that it can be listed with similar pages.
{{Improve categories}}
category links talk edit
This page needs additional or more specific categories. Please help out by adding categories to it so that it can be listed with similar articles.
{{Unreferenced category}}
category links talk edit
This article contains unreferenced categories. Please help improve this article by adding citations to reliable sources. Unsourced categories may be challenged and removed. (Learn how and when to remove this template message)
