Usage

This template is for using emoji on discussion and user pages. Don’t use it on encyclopedic articles!

Parameter Default Explanation
default or hex — Unicode number for the emoji (without U nor +). You can find it on:
  • The emoji gallery
  • List of emoji on Commons
  • List of emoji on Unicode.org

For instance, 263A is a smiling face, so {{emoji|263A}} or {{emoji|hex=263A}} becomes .

name — Human-readable name or emocode for the emoji. See Module:Emoji/data for all accepted emocodes.

For instance, {{emoji|name=grin}} becomes .

size 16 Size for the emoji in pixels.
theme one Visual theme for the emoji, chosen among:
  • android : Android Emoji
  • fx : Firefox OS Emoji
  • none: Disallow image display, and show the raw Unicode character instead. Uses size as font size in pixels.
  • noto: Noto Color Emoji Nougat
  • noto 4.4: Noto Color Emoji KitKat
  • noto 5.0: Noto Color Emoji Lollipop
  • noto 8.0: Noto Color Emoji Oreo
  • noto 9.0: Noto Color Emoji Pie
  • one: Emoji One
  • one bw: Emoji One BW
  • one v1: Emoji One v1
  • phantom: Phantom Open Emoji
  • twitter: Twitter Emoji
alt — Custom alt text for the emoji. If not set, the emoji will be used instead.

It is worth noting that some emoji sets may support some emoji sequences separated with a hyphen, e.g. {{Emoji|1F1FB-1F1E6}} → .

Examples

Code Rendered as
{{emoji|263A}}
{{emoji|263A|theme=twitter}}
{{emoji|263A|size=24}}
{{emoji|263A|theme=noto|size=36}}
{{emoji|263A|theme=fx|size=48}}
{{emoji|263A|theme=none}} ☺
{{emoji|263A|theme=none|size=36}} ☺
