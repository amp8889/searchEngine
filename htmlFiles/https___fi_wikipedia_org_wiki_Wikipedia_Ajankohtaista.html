Merkittyjen versioiden päivitystä

Merkityt versiot -lisäosa jolla suomenkielisen Wikipedian seulonta on toteutettu on ollut pitkään ylläpitämättömänä ja siinä on runsaasti teknistä velkaa. Ratkaisuna tähän lisäosasta poistetaan kokonaan käyttämättömiä ja vähän käytettyjä ominaisuuksia. Tällä pyritään siihen, että muutosten seurantaan liittyvä osa saataisiin päivitettyä kuntoon. Tekniikka-uutiset (viikko 16)

Viimeisimmät tekniset uutiset Wikimedian tekniseltä yhteisöltä. Kerro muille käyttäjille näistä muutoksista. Kaikki muutokset eivät vaikuta sinuun. Käännöksiä ovat saatavilla.

Ongelmia

  • Between 2 April and 8 April, on wikis using Flagged Revisions, the "Tämä muokkaus on kumottu" tag was not applied to undone edits. In addition, page moves, protections and imports were not autoreviewed. This problem is now fixed. [1][2]

Muutoksia myöhemmin tällä viikolla

  • MediaWikin uusi versio otetaan käyttöön testiwikeissä ja MediaWiki.orgissa 16. huhtikuuta. Se otetaan käyttöön kaikissa ei-Wikipedia-wikeissä ja joissakin Wikipedioissa 17. huhtikuuta. Se on käytössä kaikissa wikeissä 18. huhtikuuta (kalenteri). [3][4]
  • Default category sort keys will now affect categories added by templates placed in footnotes. Previously footnotes used the page title as the default sort key even if a different default sort key was specified (category-specific sort keys already worked). [5]
  • Väärinkäyttösuodattimeen lisätään uusi muuttuja page_last_edit_age. Muuttuja kertoo, kuinka monta sekuntia sitten sivuun tehtiin viimeisin muutos. [6]

Tulevia muutoksia

  • Vapaaehtoiskehittäjiä pyydetään päivittämään heidän työkalujaan ja ominaisuuksiaan tukemaan väliaikaistunnuksia. Lisätietoja.
  • Neljä tietokantakenttää poistetaan tietokantakopioista. Mutos vaikuttaa myös Quarryyn. Muutos koskettaa tauluja abuse_filter ja abuse_filter_history. Jotkut kyselyt saattavat vaatia päivittämistä. [7]
