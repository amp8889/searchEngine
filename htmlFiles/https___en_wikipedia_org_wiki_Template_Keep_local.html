Template documentation[view] [edit] [history] [purge]
This template is used on approximately 6,200 pages and changes may be widely noticed. Test changes in the template's /sandbox or /testcases subpages, or in your own user subpage. Consider discussing changes on the talk page before implementing them.
If you plan to make breaking changes to this template, move it, or nominate it for deletion, please notify Twinkle's users and maintainers at Wikipedia talk:Twinkle as a courtesy, as the standard installation of Twinkle adds and removes this template. Thank you!

Usage

{{Keep local}} will give the default request to keep the local file:

The uploader or another editor requests that a local copy of this file be kept.
This image or media file may be available on the Wikimedia Commons as Template:Keep local. While the license of this file may be compliant with the Wikimedia Commons, an editor has requested that the local copy be kept too. This file does not meet CSD F8 and should not be tagged as a Commons duplicate. If you desire to nominate it for deletion, notify the tagging editor.

If the file is on Commons under a different name, you can add {{Keep local|PAGENAME ON COMMONS}}, where PAGENAME ON COMMONS is the name of the page on Commons of the duplicate file.

The uploader or another editor requests that a local copy of this file be kept.
This image or media file is available on the Wikimedia Commons as SOMEPAGENAME. While the license of this file may be compliant with the Wikimedia Commons, an editor has requested that the local copy be kept too. This file does not meet CSD F8 and should not be tagged as a Commons duplicate. If you desire to nominate it for deletion, notify the tagging editor.

If you want to add why this file should not be deleted, you can write {{Keep local|SOMEPAGENAME|reason=add your reason here}}, though this is not required.

The uploader or another editor requests that a local copy of this file be kept.
This image or media file is available on the Wikimedia Commons as SOMEPAGENAME. While the license of this file may be compliant with the Wikimedia Commons, an editor has requested that the local copy be kept too because .... This file does not meet CSD F8 and should not be tagged as a Commons duplicate. If you desire to nominate it for deletion, notify the tagging editor.

This template categorized into Category:Wikipedia files on Wikimedia Commons for which a local copy has been requested to be kept.
