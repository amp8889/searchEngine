These templates are used for splitting articles by using ({{Split}}, {{Split section}}, etc.

For use on main page[edit]

Content split

When two or more topics are being dealt with on the same page which share the same or similar title and a disambiguation page seems appropriate use {{Split}}:

It has been suggested that this page should be split into multiple pages. (discuss)
  • Place the template at the top of the page for which the split is proposed.
  • By default, the discussion is directed to the talk page of the page on which this tag is placed. Optionally, |discuss= can be added to direct the discussion to another page. Example: {{split|discuss=Talk:Foobar#Split proposal}}, produces:
It has been suggested that this page should be split into multiple pages. (discuss)
  • Optionally, you can include suggested titles for between 2 and 20 new pages as parameters. Example: {{split|New title 1|New title 2}}, produces:
It has been suggested that this page should be split into pages titled New title 1 and New title 2. (discuss)
  • Add the date parameter |date=April 2024 to place articles in Category:Articles to be split from April 2024
Size split

When an article is too long, and it is uncertain how the article should be split, use {{Very long}}:

This page may be too long to read and navigate comfortably. Consider splitting content into sub-articles, condensing it, or adding subheadings. Please discuss this issue on the article's talk page.
  • When there is a suggested new article name for some content use {{Split}} with |New article name=:
It has been suggested that this page should be split into a new page titled New article name. (discuss)
  • When an article series is proposed, use {{Split}} with multiple titles:
It has been suggested that this page should be split into pages titled List of subject, A–M and List of subject , N–Z. (discuss)
It has been suggested that this page should be split into pages titled Subject in Africa, Subject in the Americas, Subject in Asia and the Pacific and Subject in Europe. (discuss)

When a disambiguation page needs splitting use {{Split dab}}:

It has been suggested that this disambiguation page should be split into multiple pages. (discuss)

For sections, use:

  • {{Split section}}
It has been suggested that this section be split out into another page. (Discuss)
  • {{Splitsections}} - can specify multiple destination articles
It has been suggested that sections of this page be split out into another page. (Discuss)
  • {{Move portions}} - for migrating portions of a section
It has been suggested that portions of this page be split out into another page. (Discuss)
  • {{Move section portions}} - for a less disruptive tag:
It has been suggested that portions of this section be split out into another page. (Discuss)

For use on talk page[edit]

  • {{Copied}} (for both source & destination) - {{Copied|from=|from_oldid=|to=|to_diff=|to_oldid=}}
Text and/or other creative content from Foo was copied or moved into Bar. The former page's history now serves to provide attribution for that content in the latter page, and it must not be deleted as long as the latter page exists.
  • {{Split article}} (supporting multiple splits from the same page) - {{Split article|from=|from_oldid=|to=|diff=|date=}}
    Material from Foo was split to Bar. The former page's history now serves to provide attribution for that content in the latter page, and it must not be deleted so long as the latter page exists. Please leave this template in place to link the article histories and preserve this attribution. The former page's talk page can be accessed at Talk:Foo.

During and after splitting[edit]

  • {{Being split}} - {{Being split|Destination}}
This page is currently being split.
After a discussion, consensus to split this page with Destination was found. You can help implement the split by following the instructions at Help:Splitting and the resolution on the discussion.
  • {{Cleanup split}} - {{Cleanup split|Destination}}
This article or section may need to be cleaned up or summarized because it has been split from/to Destination.

  • By default, the discussion is directed to the talk page of the page on which this tag is placed. Optionally, a "discuss" parameter can be specified to direct the discussion to another page.
  • Add the date parameter |date=April 2024 to place articles in Category:Articles to be split from April 2024

links talk edit
It has been suggested that this section be split out into another page titled New title. (Discuss)
  • Optionally, you can include a suggested title for the new page as a parameter.
  • Add the date parameter |date=April 2024 to place articles in Category:Articles to be split from April 2024
{{split sections}}
links talk edit
It has been suggested that sections of this page be split out into another page titled New title. (Discuss)
  • Place this at the top of a page for which a split of multiple sections into a new page is proposed.
  • By default, the discussion is directed to the talk page of the page on which this tag is placed. Optionally, a "discuss" parameter can be specified to direct the discussion to another page.
  • If possible, instead use {{split section}} or {{split section|new title}} in each of the sections that should be split into the new page.
  • Add the date parameter |date=April 2024 to place articles in Category:Articles to be split from April 2024
{{sub-sections}}

This section needs to be divided into subsections. Please help improve the section and read the Manual of Style for guidelines.
  • Place this at the top of one or more sections for which a split into a new section on the same page.
  • By default, the discussion is directed to the talk page of the page on which this tag is placed. Optionally, a "discuss" parameter can be specified to direct the discussion to another page.
