AutoWikiBrowser

Sorry, unregistered users cannot be granted permissions due to technical restrictions. Please create an account in order to request user account permissions.
(add request • view requests)
Shortcuts
  • WP:PERM/AWB
  • WP:RFP/AWB
Please review the AWB guide before making a request. Users are rarely approved unless they have either 250 non-automated mainspace edits or 500 total mainspace edits. You can check your non-automated contributions on XTools. Administrators are automatically approved and do not need to request access. For bots, you must first have a bot approval request approved, or in trial; link to your BRFA in the request.
This page is for requesting AWB Access for English Wikipedia only. Other projects have their own check pages and you should ask administrators there to add you.
If approved, your name will be added to the CheckPage. If you will make many or fast automated edits using AWB, you may wish to use a separate (or bot) account for that purpose; first create the account, give it a userpage declaring it to be your alternate account, then make the request using your main account.
