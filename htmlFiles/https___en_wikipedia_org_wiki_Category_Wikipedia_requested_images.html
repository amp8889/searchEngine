Fulfilling a request[edit]

If you fulfilled an image request:

  • Be sure to remove the image request template from the article's talk page.
  • Check Wikipedia:Requested pictures, to see if the request was listed there, and remove it.
  • Please ensure all image pages include copyright information. (See Wikipedia:File copyright tags).
