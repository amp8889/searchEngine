Example of box, usage instructions

{{divbox|box color|title|content}}

  • box color: see list
  • title: appears strong and centered at top
  • content: any text, including markup, for which the restrictions on parameter values are satisfied

The box has the full available width. In this example the width is reduced by the fact that the box is placed in a table.

{{Col-break|gap=1em}}

{{col-end}}

This is column 1. Text is wrapped appropriately.

This is column 2. Text is wrapped appropriately.

This is column 3. Text is wrapped appropriately.

This can be used anywhere. Creates multiple columns in a single row. You can have as many columns as you want. Text is wrapped appropriately, and a margin prevents text in one column from touching text in another column. You can use any Wiki markup within the columns created by these templates.

{{col-end}}
