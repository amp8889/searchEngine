Description

Steps to replicate the issue: Import a page on a target page that is already existing on a wiki with FlaggedRevs and is currently marked as reviewed.

What happens?: The edit for the import is not autoreviewed (see https://de.wikipedia.org/w/index.php?title=Der_Rikschamann_(1958)&action=history as an example) and needs to be reviewed manually

What should have happened instead?: The edit should be marked as autoreviewed
