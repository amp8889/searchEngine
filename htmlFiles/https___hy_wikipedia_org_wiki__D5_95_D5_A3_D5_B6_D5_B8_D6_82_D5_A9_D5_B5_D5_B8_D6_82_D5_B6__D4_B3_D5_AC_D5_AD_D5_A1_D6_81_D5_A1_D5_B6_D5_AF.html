Պաշտոնական
քաղաքականություն		 Ընդհանուր հրաժարագիր  • Դուք Վիքիպեդիան օգտագործում եք Ձեր սեփական պատասխանատվությամբ (ՎՊ:ՌԻՍԿ)  • Վիքիպեդիան բժշկական խորհուրդներ չի տալիս  • Վիքիպեդիան իրավաբանական խորհուրդներ չի տալիս  • Վիքիպեդիայի բովանդակությունը կարող է վիճարկելի լինել (ՎՊ:ՎԻՃԱՐԿԵԼԻ)  • Վիքիմեդիա Հիմնադրամի աշխատակիցների միջամտություն
Հինգ սյուներ Վիքիպեդիան հանրագիտարան է (ՎՊ:ԻՎՉ)  • Չեզոք տեսակետ (ՎՊ:ՉՏ)  • Ազատ օգտագործում (ՎՊ:ՀԻ / ՎՊ:ԱԶԱՏ)  • Էթիկական պահվածք (ՎՊ:ԷՊ / ՎՊ:ՊԱՀՎԱԾՔ)  •Չունի խիստ կանոններ (ՎՊ:ԱԲԿ)
Գործունեություն Գաղտնիության քաղաքականություն  • Ադմինիստրատորների ընտրության կանոններ (ՎՊ:ԱԸԿ)  • Կանոնների ցանկ (ՎՊ:Կ)
Մեխանիզմներ Վերահղումներ (ՎՊ:ՎՀ)  • Բոտերի կիրառման կանոններ (ՎՊ:ԲԿԿ)  • Համաձայնություն (ՎՊ:Հ / ՎՊ:ԿՈՆՍ)  • Խմբագրման հետ շրջում (ՎՊ:ԽՀՇ)  • Հետ գլորում (ՎՊ:ԳԼՈՐՈՒՄ)

Հոդվածներ
Պարունակություն Կանոններ և ուղեցույցներ (ՎՊ:ԿՈՒ)  • Վստահելի աղբյուրներ (ՎՊ:ՎԱ)  • Ապրող անձանց կենսագրություններ (ՎՊ:ԱԱԿ / ՎՊ:ԱՊՐՈՂ)  • Ցանկեր (ՎՊ:Ց / ՎՊ:ՑԱՆԿԵՐ)  • Նշանակալիություն (ՎՊ:Ն)  • Հոդվածների անվանում (ՎՊ:ՀԱ)  •Բարեխիղճ օգտագործման չափանիշներ (ՎՊ:ԲՕՉ)  • Նվազագույն պահանջներ (ՎՊ:ՆՊ)  • Հոդվածի չափ (ՎՊ:ՀՉ / ՎՊ:ԿԱՐՃ / ՎՊ:ՆՎԱԶ)
Նշանակալիություն և ԲՕՉ Երաժշտական խմբեր և կատարողներ (ՎՊ:ՆԵ / ՎՊ:ՆԵԽ)  • Զինվորականներ և ռազմական գործիչներ (ՎՊ:ԶԻՆ)  • Անիմե և մանգա (ՎՊ:ՆՅԱ / ՎՊ: ^_^)  • Վեբ-կայքեր (ՎՊ:ՎԵԲ / ՎՊ:ՎԿՆՉ)  • Պետական բարձրագույն մրցանակների ասպերներ (ՎՊ:ՊԲՄԱ)  • Ծրագրեր (ՎՊ:ԾՆ / ՎՊ:ԾՆՉ)  • Մարդիկ (ՎՊ:ԲԻՕ / ՎՊ:ՄՆ)  • Ֆուտբոլիստներ (ՎՊ:ՖՈՒՏ)  • Անիմեի և մանգայի ԲՕ (ՎՊ:ԲՕՉ-ԱԵՎՄ)  • Հորինված աշխարհների կերպարների պատկերների ԲՕ (ՎՊ:ԲՕՉ-ԿԵՐՊԱՐ)  • Ձայնային կտորների ԲՕ (ՎՊ:ԲՕՉ-ՁԱՅՆ)
Անվանում և
ձևավորում		 Աշխարհագրական անվանումներ (ՎՊ:ԱԱ) բացառություններ (ՎՊ:ԱԱ-Բ) ճշտումներ (ՎՊ:ԱԱ-Ճ)  • Դեղամիջոցներ (ՎՊ:ԴԵՂ)  • Սպորտային թիմեր (ՎՊ:ՀԱ-ՍՊՈՐՏ)  • Հոդվածներ անատոմիայի մասին (ՎՊ:ՀԱ-ԱՆԱՏՈՄԻԱ)  • Կերպարվեստի ստեղծագործություններ (ՎՊ:ՀԱ-ԿՍ)  • Ձևավորում (դրոշներ) (ՎՊ:ԴՐՈՇ)  • Լատինատառ պարունակող երաժշտական թեմատիկայի վերնագրեր (ՎՊ:ԵՐԼԱՏ)
Ոճ Բազմիմաստության փարատում (ՎՊ:ԲՓ)  • Ստուգելիություն (ՎՊ:ՍՏՈՒԳ)  • Մեջբերում (ՎՊ:ՄԵՋ)  • Մշակույթում (ՎՊ:ՄՇԱԿՈՒՅԹ)
Ձևավորում Հոդվածների ձևավորում (ՎՊ:ՀՁ)  • Ներքին հղումներ (ՎՊ:ՆՀ)  • Արտաքին հղումներ (ՎՊ:ԱՀ)  • Ծանոթագրություններ (ՎՊ:ԾՆԹ)  • Հղումներ աղբյուրներին (ՎՊ:ՀՂԱ)  • Վիքիֆիկացում  • Կատեգորիա (ՎՊ:ԿԱ)  • Բանաձևեր (ՎՊ:ԲՆՁ / ՎՊ:ԲԱՆԱՁԵՎ)  • Գույներ (ՎՊ:ԳՈՒՅՆԵՐ)  • Պատկերազարդում (ՎՊ:Պ)
Շեղումներ Մարգինալ տեսություններ (ՎՊ:ՄԱՐՏ)  • Ոչ մի սկզբնական հետազոտություն (ՎՊ:ՍՀ / ՎՊ:ՈՄՍՀ)  • Մոլորեցումներ (ՎՊ:ՄՄ)  • Վանդալություն (ՎՊ:Վ / ՎՊ:ՎԱՆԴ)  • Արագ ջնջման չափանիշներ (ՎՊ:ԱՋՉ)  • Հոդվածների լցոնում (ՎՊ:ԼՑՈՒՄ)  • Որբ-հոդվածներ (ՎՊ:ՈՀ / ՎՊ:ՈՐԲ)  • Սպամ (ՎՊ:ՍՊԱՄ)  • Էջերի ջնջում (ՎՊ:ԷՋ / ՎՊ:ԷՋՋ)  • Հոդվածների անվանափոխում (ՎՊ:ԱՆՎ)

Վարվելակարգի
նորմեր		 Քաղաքավարություն (ՎՊ:Ք)  • Ունեցեք բարի մտադրություններ (ՎՊ:ՈՒԲՆ)  • Մի վիճեք վանդալների հետ (ՎՊ:ՄՎՎՀ)  • Մի կառչեք նորեկներից (ՎՊ:ՄԿՆ)  • Մի խաղացեք կանոնների հետ (ՎՊ:ՄԽԿ)  • Մի վիճեք վանդալների հետ (ՎՊ:ՄՎՎՀ)  • Մի հասցրեք աբսուրդի (ՎՊ:ԱԲՍՈՒՐԴ)  • Ոչ մի անձնական վիրավորանք (ՎՊ:ԱՎ)  • Մի սպառնացեք դատական հայցերով (ՎՊ:ԴԱՏ)  • Համարձակ եղեք (ՎՊ:ՀԵ / ՎՊ:ՀԱՄԱՐՁԱԿ)  • Ստորագրեք քննարկման էջերում (ՎՊ:ՍՏ)  • Չնչին փոփոխություններ (ՎՊ:Չ / ՎՊ:ՉՆՉԻՆ)  • Վիքիպեդիա:Վիքինախագծեր (ՎՊ:ՆԱԽԱԳԻԾ)
Նախագծային «Շաբաթվա հոդված» նախագիծ (ՎՊ:ՇՀԿ)
Շեղումներ Բաց պրոքսի-սերվերների օգտագործման անթույլատրելիություն (ՎՊ:ԲՊՕԱ / ՎՊ:ՊՐՈՔՍԻ)  • Խամաճիկություն (ՎՊ:Խ)  • Երեք հետշրջման կանոն (ՎՊ:ԵՀԿ / ՎՊ:3Հ / ՎՊ:3ՀԿ)  • Ապակառուցողական վարքագիծ (ՎՊ:ՈՉԿԱՌ)  • Խմբագրական պատերազմ (ՎՊ:ՊԱՏԵՐԱԶՄ / ՎՊ:ԽՊ)
Կարգապահություն Կոնֆլիկտների լուծում (ՎՊ:ԿԼ)  • Արգելափակում (ՎՊ:ԱՐԳ / ՎՊ:ԱՐԳԵԼՔ)  • Էջերի պաշտպանության կանոններ (ՎՊ:էՊԿ)  • Էջերի մասնակի պաշտպանություն (ՎՊ:ԷՄՊ)
