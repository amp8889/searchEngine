The following tags should be placed on top the pages or sections which require merging.

Tag Text that will be shown (and usage notes)
{{Afd-merge from}}
links talk edit 		Please move this tag to the talk page:
The article [[{{{1}}}]] was nominated for deletion. [[Wikipedia:Articles for deletion/{{{2}}}|The discussion]] was closed on {{{3}}} with a consensus to merge the content into Wikipedia:Template index/Merging. If you find that such action has not been taken promptly, please consider assisting in the merger instead of re-nominating the article for deletion. To discuss the merger, please use this talk page.
Do not remove this template after completing the merger. A bot will replace it with {{afd-merged-from}}.
  • Place this on the article's talk page
{{Afd-merge to}}
links talk edit
This article was nominated for deletion. The discussion was closed on 21 April 2024 with a consensus to merge the content into the article New. If you find that such action has not been taken promptly, please consider assisting in the merger instead of re-nominating the article for deletion. To discuss the merger, please use the destination article's talk page. (April 2024)
{{merge}}
category links talk edit
It has been suggested that this page be merged with Other page. (Discuss)
  • Place this at the top of both pages or sections that are to be merged. Use the other page's name as the parameter. A link will be generated automatically. Separate additional pages with a vertical pipe; up to 20 pages can be listed.
  • By default, the discussion is directed to the talk page of the page on which this tag is placed. Optionally, discuss can be specified to direct the discussion to another page, see example of discuss usage.
  • If it is known which page or section is to receive the merged material, use the tags {{merge from}} or {{merge to}} instead.
  • Add the date parameter |date=April 2024 to place articles into Category:Articles to be merged from April 2024
{{merge from}}
links talk edit
It has been suggested that Source and Talk page be merged into this page. (Discuss)
  • If the proposed source is an entire page, place this at the top of the page or section that is to receive the merged material.
  • Place the tag {{merge to|Destination page}} at the top of the source page or section.
  • If a merger of multiple pages into a single page is being proposed, use {{merge from|Source 1|Source 2|Source 3}}.
  • By default, the discussion is directed to the talk page of the page on which this tag is placed. Optionally, the discuss parameter can be specified to direct the discussion to another page, see example of discuss usage.
  • If it is not known which page or section is to receive the merged material, use the tag {{merge|Other page}} instead.
  • Add the date parameter |date=April 2024 to place articles into Category:Articles to be merged from April 2024
{{merge to}}
links talk edit
It has been suggested that this page be merged into Destination page and Talk page. (Discuss)
  • Place the tag {{merge to|Destination page}} at the top of the page or section that is to have its content transferred to another page.
  • If the proposed source is an entire page, place the tag {{merge from|Source page}} at the top of the page or section that is to receive the merged material.
  • If the proposed source is one specific section of page, place the tag {{merge from|Source page#Source section}} at the top of the page or section that is to receive the merged material.
  • Add the date parameter |date=April 2024 to place articles into Category:Articles to be merged from April 2024
{{merged-from}}
links talk edit
The contents of the Source page page were merged into Wikipedia:Template index/Merging. For the contribution history and old versions of the redirected page, please see its history; for the discussion at that location, see its talk page.
{{merged-to}}
links talk edit
The contents of the Wikipedia:Template index/Merging page were merged into Destination page. For the contribution history and old versions of the merged article please see its history.
{{merging}}
links talk edit
This page is currently being merged.
After a discussion, consensus to merge this page with Other page was found. You can help implement the merge by following the instructions at Help:Merging and the resolution on the discussion.
  • Place this at the top of a source page for which a merger has begun (but has not yet been completed).
  • Add the date parameter |date=April 2024 to place articles into Category:Articles to be merged from April 2024
{{Split portions}}
links talk edit
It has been suggested that portions of this page be split out into pages titled Destination page and Talk page. (Discuss)
