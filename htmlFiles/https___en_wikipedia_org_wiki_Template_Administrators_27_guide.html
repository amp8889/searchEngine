Shortcut

Use {{Administrators' guide|state=collapsed}} to show this template in its collapsed (hidden) state.
Use {{Administrators' guide|state=expanded}} to show this template in its expanded (uncollapsed) state. (This is the default.)
Use {{Administrators' guide|state=autocollapse}} to show this template in its collapsed (hidden) state only if there is another template of the same type on the page.
