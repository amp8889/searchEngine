De unde provine fișierul? (Apăsați pe legătura potrivită)

  • Îmi aparține în întregime și doresc să-l încarc sub o licență liberă
    • Este o imagine a unui monument istoric din România (mai multe despre Lista Monumentelor Istorice)
    • Este o imagine a unei opere de artă sau arhitectură din secolele XX sau XXI din România
    • Alte fișiere care îmi aparțin
  • Aparține altcuiva, care și-a dat permisiunea pentru a fi eliberat sub o licență liberă
  • A fost încărcat pe Flickr de către altcineva (mai multe despre conținutul de pe Flickr)
  • Este o imagine promoțională dintr-o reclamă, sau o altă sursă promoțională
  • Este o copertă sau o altă pagină dintr-o carte, album, DVD, ziar, revistă, sau o altă astfel de sursă
  • Este o captură de ecran luată dintr-un film, program TV, joc pe calculator, pagină web, program pe computer, video, sau o altă astfel de sursă
  • Este imaginea unui timbru poștal, sau a unei monede
  • Este o imagine de pe o pagină web
  • Este de altundeva
  • Nu știu cine este autorul sau nu știu ce fel de licență este potrivită

Pentru utilizatorii experimentați:

  • Știți deja care e licența potrivită? Mergeți direct la formularul de trimitere.
  • Aveți pregătit formatul {{information}} (de exemplu, unul generat de Flinfo)? Încercați formularul simplu de încărcare.

Vedeți lista completă a formatelor de licențiere pentru mai multe țări ale lumii. Prima secțiune a acestei pagini conține formate generice pentru fișierele aflate în domeniul public.
