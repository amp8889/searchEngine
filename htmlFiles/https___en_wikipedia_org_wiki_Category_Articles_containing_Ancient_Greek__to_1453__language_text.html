This category contains articles with Ancient Greek-language text. The primary purpose of these categories is to facilitate manual or automated checking of text in other languages.

This category should only be added with the {{Lang}} family of templates, never explicitly.

For example {{Lang|grc|text in Ancient Greek (to 1453) language here}}, which wraps the text with <span lang="grc">. Also available is {{Lang-grc|text in Ancient Greek (to 1453) language here}} which displays as Ancient Greek: text in Ancient Greek (to 1453) language here.
