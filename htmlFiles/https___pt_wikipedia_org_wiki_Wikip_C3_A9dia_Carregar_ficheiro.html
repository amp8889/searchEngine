Origem: Wikipédia, a enciclopédia livre.
Nota: Esta página serve para auxiliar o carregamento de ficheiros multimédia na Wikipédia lusófona.

Tenciona carregar algum ficheiro de conteúdo livre? Carregue os ficheiros de licença livre no Wikimedia Commons, seguindo estas instruções.
Itens carregados no Commons podem ser utilizados na Wikipédia e noutros locais, ajudando a criar um repositório de imagens e multimédia livres. Se possuir login unificado (ajuda), exerça-o no Commons. Caso queira carregar uma imagem de acordo a política de uso restrito, siga estas instruções.

Qual é a origem da obra que quero carregar?

  • É inteiramente trabalho próprio
  • É do Flickr
  • É de outro projeto Wikimedia (outra Wikipédia, Wikilivros, Wikinotícias, etc)
  • É uma obra de outra origem
  • Não sei quem é o autor
