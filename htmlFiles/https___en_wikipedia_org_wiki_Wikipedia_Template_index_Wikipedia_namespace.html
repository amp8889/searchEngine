The Wikipedia:Project namespace includes Wikipedia official policies and guidelines, process pages, discussion pages, optional essays, informative pages, humorous and historical pages.

For the rules concerning the use of these tags, see WP:Policies and guidelines.

Guideline category should be one of:

  • behavioral guideline
  • content guideline
  • deletion guideline
  • editing guideline
  • naming convention
  • notability guideline
  • style guideline
 Example for naming convention as <guideline category>:
This guideline documents an English Wikipedia naming convention.
It is a generally accepted standard that editors should attempt to follow, though occasional exceptions may apply. Any substantive edit to this page should reflect consensus. When in doubt, discuss first on the talk page.

How-to and information

Further information: WP:INFOPAGES
What to type What it makes Notes
{{Information page}}
links talk edit
This is an information page.
It is not one of Wikipedia's policies or guidelines; rather, its purpose is to explain certain aspects of Wikipedia's norms, customs, technicalities, or practices. It may reflect differing levels of consensus and vetting.
Shortcut
  • WP:SHORT
 The banner template for administration pages that are more just informational and non-opinionated like WP:GOV.
{{Wikipedia how to}}
links talk edit
This help page is a how-to guide.
It details processes or procedures of some aspect(s) of Wikipedia's norms and practices. It is not one of Wikipedia's policies or guidelines, and may reflect varying levels of consensus and vetting.
Shortcut
  • WP:SHORT
 The banner template for administration pages that are more directly instructional and non-opinionated like H:EDIT.
{{supplement|pages=[[Wikipedia:Policy]] page}}
This is an explanatory essay about the Wikipedia:Policy page.
This page provides additional information about concepts in the page(s) it supplements. This page is not one of Wikipedia's policies or guidelines as it has not been thoroughly vetted by the community.
Explanatory essay about the Wikipedia:Policy page
 Use this template carefully, only when there is a well-established consensus at the relevant policy or guideline page to link to the administration page in question from the policy or guideline like WP:BRD.
{{Process header}}
links talk edit
Wikipedia namespace
A blank template meant to be populated and used at the top of a page in the Wikipedia and help namespace.

Historical and under development

Further information: WP:HISPAGES and Wikipedia:Template messages/Maintenance
What to type What it makes Notes
{{historical}}
links talk edit
This page is currently inactive and is retained for historical reference.
Either the page is no longer relevant or consensus on its purpose has become unclear. To revive discussion, seek broader input via a forum such as the village pump.
For a former policy, use {{Historical|type=policy}}. For a former guideline, use {{Historical|type=guideline}}
{{superseded|[[New page]]}}
This Wikipedia page has been superseded by New page and is retained primarily for historical reference.
 For process and infomation pages that have been superseded by other pages.
{{WikiProject status|Defunct}}
This WikiProject is defunct.
Consider looking for related projects for help or ask at the Teahouse.
If you feel this project may be worth reviving, please discuss with related projects first. Feel free to change this tag if the parameters were changed in error.
  • Guide to WikiProjects
  • Directory of WikiProjects
 Set inactive=yes parameter from {{WikiProject status}}.
{{Essay in development}}
links talk edit
This essay is in development.
It contains the advice or opinions of one or more Wikipedia contributors.
Essays may represent widespread norms or minority viewpoints. Consider these views with discretion, especially since this page is still under construction.
 This template is for essay pages actively undergoing a major edit.
{{Under construction}}
links talk edit
This project page or section is in a state of significant expansion or restructuring. You are welcome to assist in its construction by editing it as well. If this project page has not been edited in several days, please remove this template.
If you are the editor who added this template and you are actively editing, please be sure to replace this template with {{in use}} during the active editing session. Click on the link for template parameters to use. This page was last edited by Brennan Everette (talk | contribs) 3 months ago. (Update timer)
 This template is for administration pages actively undergoing a major edit.
{{In use}}
links talk edit
This project page is actively undergoing a major edit for a short while. To help avoid edit conflicts, please do not edit this page while this message is displayed.
This page was last edited at 12:26, 7 January 2024 (UTC) (3 months ago) – this estimate is cached, update. Please remove this template if this page hasn't been edited for a significant time. If you are the editor who added this template, please be sure to remove it or replace it with {{Under construction}} between editing sessions.
 Placed at the top of an administration page you are actively editing for a short period of time
{{WP Future}}
links talk edit
This page is not yet active and is retained for future deliberations. Either the page is not yet relevant, or its purpose is dependent on future events or inventions. If events have occurred that mean we now need policy in this area it may be worth seeking broader input via a forum such as the village pump proposals page.
 Pages using this template will be placed into Category:Inactive project pages unless the parameter category=no is given
{{Freq}}
links talk edit
The following is a feature request for Wikipedia, the details of which may still be in development or under discussion. As a whole, this request should be discussed with the developers, who will decide whether or not to act on it.
 This template is meant to be used at the top of a page in the Wikipedia and help namespace.
{{Brainstorming}}
links talk edit
This idea is in the brainstorming stage.
Feel free to add new ideas; improve, clarify and classify the ideas already here; and discuss the merits of these ideas on the talk page.
 This template is meant to be used at the top of a page in the Wikipedia and help namespace currently being developed.

Main page: Wikipedia:Template messages/Cleanup
What to type What it makes Notes
{{Disputed tag}}
links talk edit
This page's designation as a policy or guideline is disputed or under discussion. Please see the relevant talk page discussion for further information.
 template to dispute whether a page (or a section on a page) has properly been accorded policy or guideline status.
{{Under discussion}}
links talk edit
This page is the subject of a current discussion. Please feel free to join in. This doesn't mean that you may not be bold in editing this page, but that it would be a good idea to check the discussion first.
 If the discussion concerns a proposed change to the status of a policy or guideline page then you may use the status parameter (see Policies and guidelines above).

 A blank template used at the top of administration pages that can be populated manually.

links talk edit
This Wikipedia page needs to be updated. Please help update this Wikipedia page to reflect recent events or newly available information. Relevant discussion may be found on the talk page.
 Used to denote process, how-to and information pages that contain old or out-of-date information that needs updating.

{{humor}}
links talk edit
This page contains material that is kept because it is considered humorous.
Such material is not meant to be taken seriously.
 For use on Wikipedia and User namespaces pages that are not serious enough for publication in the mainspace.
{{humorantipolicy}}
links talk edit
This page is intended as humor. It is not, has never been, nor will ever be, a Wikipedia policy or guideline.
Rather, it illustrates standards or conduct that are generally not accepted by the Wikipedia community.
 Humor pages that spoof and/or may have been proposed as policies or guidelines.
{{Humorous essay}}
This is a humorous essay.
It contains the advice or opinions of one or more Wikipedia contributors and is made to be humorous. This page is not one of Wikipedia's policies or guidelines, as it has not been thoroughly vetted by the community. Some essays represent widespread norms; others only represent minority viewpoints. This essay isn't meant to be taken seriously.
 For humorous essays in both the Wikipedia and User namespaces.
